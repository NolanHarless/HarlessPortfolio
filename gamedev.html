Game Development University Project

Project Scope

I was tasked by my instructor to create a one-screen platformer video game using a video game development tool called GoDot. The character had to jump through a blocky 2-dimensional world to reach the finish using their keyboard controls.

What I learned

The game development course taught me a lot about many different game design principles as well as some great programming skills. I also learned how to create concise and frequent commit messages to GitHub.

Most Proud Of

I am most proud of the artwork that I created using Adobe Photoshop and Illustrator.
